# ⚒️ Getting Started

## Installation

Requires **Python 3.10** or above.

To install, run:

```
pip install infinite-craft
```

To update, run:

```
pip install -U infinite-craft
```

**NOTE:** If `pip` is not on PATH, you can use:

* `python3 -m pip` (for Linux/MacOS)
* `python -m pip` (for Windows)
